##Nordic nRF52840 Smartwatch

Tools:
- nRF5 SDK 16.0.0
- CMake
- GCC
- FreeRTOS

Components:
- [X] TI DRV2605L Haptic Motor Driver
    - [X] I2C/TWI
    - [X] PWM - 100kHz 
- [ ] BLE
    - [ ] ANCS
    - [ ] CTS
    - [ ] Battery
- [ ] STM LIS3DH or Bosch BMI160
- [ ] Heart Rate Sensor
- [ ] Display
    - [ ] LittlevGL Library

Based on the project template at [nRF5-cmake-scripts-example](https://github.com/NRB-Tech/nRF5-cmake-scripts-example).
